# SAUSAU_Valorant_Projekat
Projekat iz predmeta Softverski algoritmi u sistemima automatskog upravljanja

Projekat "Анализа мете и предвиђање мечева на Valorant е-спорт турнирима." 
Mateja Stojišić RA180/2023

Ovaj repozitorijum sadrži dva dokumenta i tri foldera:
  - Dokumentacija_projekta.docx: Za pročitati dokle je postignuto sa projektom samim
  - Specifikacije projekta.docx: Unaprijed definisane specifikacije projekta (cilj)
  - folder dataset: Sadrži kompletan dataset korišten za treniranje i testiranje ML modela kao i fajlovi confusion_matrix, poređenje_algoritama, top_agenti_pick_rate, monte_carlo i evolucija_agenata kao rezultate pokrenutog koda
  - folder kod: Sadrži london.py fajl koji pokreće model i vrši deterministički i monte carlo simulaciju za predviđanje pobjednika mečeva na Valorant turniru VCT London Masters 2026; agenti.py analizu mete i odabiranja agenata od 2021. do 2026.
  - folder rezultati: Svi rezultati oba fajla su smješteni ovdje.
